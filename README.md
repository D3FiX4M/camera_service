# camera_service
### Задача:
 - Создание GRPC сервиса для передачи телеметрических данных и потока изображений с любой веб камеры к клиентскому приложению.
 - Сжатие изображения

### Примечание:
- настоящая вебкамера заменена на синтетическую генерацию изображения.
